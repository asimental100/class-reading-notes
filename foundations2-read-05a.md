Heroku is neat.

Deploying an app with Heroku (in the terminal steps below)
  1. heroku create
  2. git push heroku master
  3. heroku ps:scale web=1
  4. heroku open
  a. Can view heroku related events by using: "heroku logs --tail" to view logs.

The 2nd article basically just shows the above deploying steps being actually utilized.
